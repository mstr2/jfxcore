Contains style themes for JavaFX controls.

This package contains the built-in themes that are shipped with JavaFX:

  1. {@link javafx.scene.control.theme.CaspianTheme Caspian}, which is the default theme for JavaFX 2.X
  2. {@link javafx.scene.control.theme.ModenaTheme Modena}, which is the default theme for JavaFX 8.X

Style themes can be applied using the {@link javafx.application.Application#setStyleTheme} method:


    public class App extends Application {
        @Override
        public void start(Stage primaryStage) {
            setStyleTheme(new CaspianTheme());

            primaryStage.setScene(...);
            primaryStage.show();
        }
    }
